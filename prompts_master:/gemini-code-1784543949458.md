# Step 5: Project Financials, Loans, Scans & Lien Waiver Enforcement
Goal: Establish the full financial breakdown sheet, custom loan calculator, receipt uploads, and compliance manager.

## Task Details
1. **Financial Hero Panel:**
   - Large colored display highlighting overall health status (On Budget vs Over Budget) based on true total expenditures.
   - Live schedule meter comparing Work Progress % against Elapsed Project Time % to call out delays (>10% variance).
2. **Loan Financing Configurator:**
   - Implement three distinct engine models: Interest-Only (Hard money formula: `principal * APR / 12`), Amortized Principal & Interest, and Manual overrides.
   - Track Accrued Interest daily and inject it directly into the True Project Cost metrics.
3. **Receipt Box & AI Simulated Processor:**
   - List uploaded invoices with metadata. Provide a manual "AI Scan" action button that populates vendor name, total cost, and item breakdowns.
4. **Transaction History & Lien Waiver Compliance:**
   - Display a chronological ledger of payments.
   - **Enforcement Rule:** Every single financial ledger entry with a category of 'labor' or 'material' that exceeds $600 must automatically switch `lienWaiverRequired` to true.
   - Show a prominent red warning label `[WAIVER NEEDED]` next to the item. Turn to a green check label once a file URL is attached.

## Definition of Done
- Loan interest computations operate correctly. Expense thresholds trigger lien waiver compliance tracking.
- Progress documented in `zeadim/05_financials.md`, pushed to GitHub. Stop and wait for user approval.
