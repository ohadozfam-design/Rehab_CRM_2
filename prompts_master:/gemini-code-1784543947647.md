# Step 4: Advanced Scope of Work (SOW) Management View
Goal: Build the core 3-phase interactive SOW checker and builder.

## Task Details
1. **3-Phase Layout:**
   - Group items strictly into fixed phases: Phase 1 (Within Walls), Phase 2 (Surface Work), Phase 3 (Finishes).
   - Phase headers must show aggregated completeness %, current spent vs budget, and calculated deadline countdowns.
2. **Tri-State Checkbox System:**
   - State 0: Not Started (Empty box).
   - State 1: In Progress (Amber partial block, triggers "50% DONE" badge, styles row background).
   - State 2: Completed (Green check box, strikes through text, marks approved timestamp).
3. **Inline Interactive Table:**
   - Columns: Status | Item Description & Note | Labor Cost | Material Cost | Total | Actions.
   - Dynamic Layout Adjustment: If NO items inside a specific phase contain material costs, completely hide the Material column and stretch the Item Description field.
   - Variance Badges: Compare current inputs against original baseline costs. Render Red Up Arrow badge for budget increases, Green Down Arrow badge for savings.
4. **Detailed Panel Expandable:** Clicking an item opens inline form to assign a worker, add long text notes, or post comments.

## Definition of Done
- Tri-state tracking updates calculations instantly. Table layouts adjust dynamically per phase properties.
- Progress documented in `zeadim/04_sow_module.md`, pushed to GitHub. Stop and wait for user approval.
