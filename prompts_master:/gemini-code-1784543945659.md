# Step 3: Main Dashboard & 5-Step Project Wizard
Goal: Implement the primary project management home screen and the complex creation wizard.

## Task Details
1. **Main Dashboard:**
   - Top metrics bar displaying 4 KPI summary cards (Active Projects, Total Spent, Avg. Completion %, Completed).
   - Sticky Red Critical Alert banner containing dismissible alert items if any visible project is over budget or past deadline.
   - Filter/Search bar with live searching and Kanban columns layout vs Grid layout toggle.
   - 3-Column Kanban board: "Active", "In Progress", "Completed". Cards show address, dynamic progress bars, budget usage metrics, and target deadlines.
2. **5-Step New Project Wizard Modal:**
   - Step 1: Basics (Name, Address parser that splits text into City/State automatically).
   - Step 2: Team & Financing (Contact address book dropdown picker for managers/contractors).
   - Step 3: Phase Deadlines (Set specific dates for Phase 1, 2, and 3).
   - Step 4: SOW Input Mode Selector (Build manually, paste raw text, or upload file).
   - Step 5: Final Review & Save.

## Definition of Done
- Projects render accurately inside the Kanban column system. Wizard adds fully functional projects to state.
- Progress documented in `zeadim/03_dashboard_wizard.md`, pushed to GitHub. Stop and wait for user approval.
